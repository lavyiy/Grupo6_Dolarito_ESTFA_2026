# Dolarito - Plataforma de Consulta de Cotizaciones Multidivisa

## Descripción del Proyecto

**Dolarito** es una aplicación web para la consulta en tiempo real de cotizaciones de múltiples divisas tradicionales y criptomonedas, con soporte para diferentes tipos de cambio por moneda (oficial, paralelo, turista, MEP, CCL, etc.).

La plataforma permite a los usuarios:
- Realizar seguimiento personalizado de las cotizaciones de su interés
- Mantener un historial de consultas
- Visualizar gráficos de evolución histórica
- Realizar conversiones de valores con tipos de cambio actualizados
- Recibir notificaciones opcionales
- Consultar información de mercado bursátil según la moneda base seleccionada

**Objetivo principal**: Ofrecer una herramienta completa, moderna y fácil de usar para seguir el mercado cambiario argentino y mundial.
